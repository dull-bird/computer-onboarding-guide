写给刚拿到新电脑的你

从不会打字，到看懂 AI Agent

这台电脑是你的新起点。这份指南按时间顺序排好了顺序：先学会用电脑，再理解电脑是什么，再动手写代码，最后弄懂现在很火的 AI 到底在干什么。不用着急，一步一步来就好。

ROADMAP

五个阶段的学习路线

1
第 1 周

先学会「操作」电脑

打字指法、文件夹与路径概念、Win11 常用快捷键、输入法与截图。每天 20-30 分钟练打字，不用赶进度，肌肉记忆最重要。

2
第 2-3 周

搭好工具环境

装好浏览器、效率工具、密码管理器和笔记软件，养成「用工具而不是硬扛」的习惯。

3
第 1 个月

理解计算机是什么

CPU / 内存 / 硬盘的关系、操作系统在做什么、二进制与文件本质、网络基础。推荐哈佛 CS50 中文字幕版。

4
第 2 个月

编程真正入门

从 Python 零基础学起：变量、循环、函数、列表。做几个小项目：算术小游戏、词频统计、命令行小工具。

5
长期坚持

理解 AI 与 Agent

用可视化建立数学直觉，理解大模型原理，再动手用 Kimi Code 做真实小任务——整理文件、写小脚本、自动化重复劳动。想直接上手？看 Agent 实战教程。

TYPING

打字指法练习

中文 + 英文

金山打字通

最经典的中文打字教程网站，拼音、五笔、英文都能练，界面对新手友好。先用它把键位、指法过一遍。

纯英文

keybr.com

英文盲打，难度会自动适应你的水平。练熟后打代码里的英文变量名会顺手很多。

游戏化

typing.com

游戏化程度更高，适合练久了觉得枯燥的时候换换口味。

SOFTWARE

Windows 装机软件清单

效率工具

PowerToys微软官方效率工具箱：窗口分屏、批量重命名、快速启动，一个顶好几个杂牌软件
7-Zip解压缩，干净不带广告
Win + V系统自带剪贴板历史，够用了先不用装 CopyQ
Win + Shift + S系统自带截图，需要长截图/标注再考虑 ShareX

账号与笔记

Bitwarden免费开源密码管理器，越早养成一人一密码的习惯越好
OneNote系统自带，零学习成本，先用它记笔记

浏览器 · Chrome 插件

uBlock Origin广告拦截，网页干净很多
沉浸式翻译中英对照翻译，看英文技术文档时很有用
BILIBILI

值得关注的 UP 主

小甲鱼

「零基础入门学习 Python」系列，讲得极其细致耐心，专为零基础设计，国内最经典的 Python 入门教程之一。

CS50 中文字幕版

B 站搜「CS50 中文」。哈佛最经典的计算机科学导论课，从二进制讲到网页，节奏适合完全新手。

3Blue1Brown 中文精译版

数学可视化神作，「线性代数的本质」「神经网络」系列能建立起对 AI 底层数学的直觉。

李沐

「跟李沐学AI」，讲深度学习、带读论文，眼界很好，可以先收藏，编程基础打好一点再系统看。

何同学

科技测评头部 UP，视频制作水准高，用来培养对科技产品的兴趣和审美。

CAUTION

新手要避开的坑

装软件时多留一个心眼

  • 不要装「电脑管家」「驱动精灵」「XX急速下载器」这类国产全家桶和第三方下载站，容易夹带流氓软件
  • Windows 11 自带的 Defender 杀毒够用，不需要额外装杀毒软件
  • 安装向导里的「附加安装」「推荐软件」默认要取消勾选，这是最容易中招的地方
  • 尽量从官网或微软商店下载软件，不要在搜索引擎的广告链接里下载
